# AI Configuration Files

This directory contains AI assistant configuration and development planning files.

**For Client/Operations Team**: You can safely ignore this directory. These files were used during development with AI coding assistants.

**For Developers**: 
- `.cursorrules` - Cursor AI coding standards and rules
- `migrate.md` - Production migration plan
- `CURSOR_INSTRUCTIONS.md` - Step-by-step migration execution instructions

These files are development artifacts and are not required for running or deploying the application.
